UNIDAD III

Requisitos para montar un servidor web

3.1 Diseño y usabilidad

Aunque el servidor web es una infraestructura técnica, su configuración puede influir en el diseño y la usabilidad del sitio alojado. Un servidor bien configurado permite tiempos de carga rápidos, lo que es crucial para la experiencia del usuario (UX). La optimización del servidor ayuda a manejar archivos multimedia, mejora el rendimiento en páginas web dinámicas y permite un diseño responsive que se adapta a diferentes dispositivos. Además, el servidor debe soportar tecnologías clave como HTML5, CSS3 y JavaScript moderno, esenciales para un diseño visual avanzado. La usabilidad también depende de la seguridad del servidor, que debe incluir HTTPS y protección contra ataques (como DDoS) para una experiencia segura y fluida.



3.2 Desarrollo y tecnología web

El desarrollo web requiere tecnologías que deben ser compatibles con el servidor web. Por ejemplo, si el proyecto está basado en PHP, Node.js o Python, el servidor debe admitir estos entornos. El sistema operativo juega un papel clave: servidores basados en Linux (como Ubuntu) son comunes por su estabilidad y soporte para Apache, NGINX o servidores LAMP/LEMP. Además, tecnologías modernas como Docker pueden facilitar el despliegue mediante contenedores. Otros elementos esenciales incluyen la gestión de bases de datos (MySQL, PostgreSQL) y configuraciones avanzadas, como el manejo de APIs REST y sistemas de autenticación seguros.
